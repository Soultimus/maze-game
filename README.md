# Maze Game
A DDA-based raycaster built with Monogame where you can explore a randomly generated maze to find an exit

## Screenshots
![Demo](Content/screenshots/sc1.png)

## Controls
- `W, A, S, D`: Move Around
- `L & R Arrow Keys`: Rotate Camera
- `M`: Show map on console
